Currently this repository contains tasks for C++ Programming Exam.

## Contents

## How to use
1) Copy project
2) In every task folder there is **main.cc**. You should write your solution there. If code compiles, than task is solved correctly. If you think that tests(task) is incorrect, consider to make an issue.
3) If you have your own task, make a pull request with your task, and make an issue.
